Magyar szógenerátor API

Leírás

Tudhatjuk, hogy a magyar nyelv igazán nagy,egyedien teremtett szókicset tartalamaz, de még is arra kapjuk a fejünket hogy minden egyes új generáció magával ránt pár újonnan alkotott modern szavakat. Sejteni lehet már a cím alapján, hogy ez a program mit is tud pontosan. Minden további szűrő feltétellen tudunk szavakat generálni a saját felhasználásunkra. Minden bizonnyal az összes eddigi magyar szavakból tudunk véletlenszerűen válogatni. A kezelő felület letisztúlt, könnyen használható és senkinek nem fog gondot okozni a használata. Magáról a szolgáltatásról a lentebbi dokumentációban tudtok olvasni!

Használati token generálás:

A használati token IP címhez van rendelve és minden tokennek 1 hónapos lejárati dátuma van. 1 hónap múlva az adatbázsból törlődik. A lentiekben az API-t tudjátok használni az automata újra generáláshoz.

Egy IP címhez csak 1 token tartozhat

Ebben a dokumentációban részletes leírást kaptok arról, hogy hogyan kell használni a szolgáltatást. Jelenleg egy végpont létezik az API-hoz, a többi még fejlesztés alatt áll.

Token újragenerálás

$
curl -X GET https://random-magyar-words-api.herokuapp.com/generate_token

Leírás

Ha erre a végpontra küldesz egy kérést akkor az API a jelenleg érvényes használati tokent fogja vissza küldeni (ha érvényes). Ha már lejárt akkor egy teljesen újat fog generálni a küldő IP címével.
